# App Name:
MiniKeePass — Gestionnaire de mot de passe sécurisé

# Description:
MiniKeePass fournit un stockage sécurisé de vos mots de passe sur votre appareil iOS compatible avec KeePass.

- Voir, éditer, et créer des fichiers KeePass 1.x and 2.x
- Rechercher des entrées grâce à la barre de recherche intégrée comme dans l'app Mail.
- Support des fichiers Kee Pass
- Importer/exporter des fichiers vers et depuis Dropbox à l'aide de l'application officielle Dropbox ou toute autre application de stockage dans le cloud
- Copie des entrées dans le presse-papiers pour une saisie facilitée
- Possibilité d'ouvrir des sites web dans Safari pendant que MiniKeePass tourne en tâche de fond
- Empêche un accès non-autorisé à MiniKeePass avec un code secret
- Possiblité d'effacer le presse-papier après avoir défini une durée
- Génère de nouveaux mots de passe
- Navigateur web intégré avec auto-remplissage du nom d'utilisateur et du mot de passe

Le code source de MiniKeePass est disponible pour des vérifications de sécurité ou pour contribuer sur GitHub.
